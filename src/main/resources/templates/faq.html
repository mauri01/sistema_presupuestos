Preguntas Frecuentes

¿Por que recibo error 401?

Ciertas pantallas estan diseñadas para el usuario ADMIN.
Revise el Rol del usuario para validar si es ADMIN
No olvides consultar la GUIA por cualquier cosa.
Puedes utilizar el usuario ADMIN por defecto que figura en la guia WORD de Drive

¿Cual es el formato de Excel?

Para poder ingresar una lista de precios es necesario incluir un archivo excel.
Ademas es importante que contenga las columnas:
| id | nombre | precio | unidad |
